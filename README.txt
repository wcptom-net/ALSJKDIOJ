1. 程序簡介
Auto-PPT Generator 是一個基於 Python 的自動化工具，旨在從純文字檔案生成具有主題樣式的 PowerPoint 幻燈片。該工具結合了人工智能（OpenAI API）和 GUI（PyQt5），用戶可以輕鬆選擇來源資料夾、主題顏色，並生成美觀的 PPT 文件。

2. 技術棧
Python 3.8+：程序的主要開發語言。
python-pptx：用於生成和操作 PowerPoint 文件。
PyQt5：用於構建現代化的 GUI。
OpenAI API：用於將純文字內容轉換為大綱。
CSS：用於美化 GUI 的樣式。
3. 程序用途
自動化生成 PPT：

從純文字檔案中提取內容，生成大綱。
將大綱轉換為具有主題樣式的 PowerPoint 幻燈片。
可視化操作：

用戶可以通過 GUI 選擇來源資料夾和主題顏色。
支持進度條顯示處理進度。
靈活性：

支持多種顏色主題。
可輕鬆修改代碼以適應不同需求。

6. 代碼結構
generate_outline_from_text：
從純文字文件中讀取內容，並通過 OpenAI API 生成大綱。
create_ppt_from_outline：
將生成的大綱轉換為 PowerPoint 幻燈片，並應用主題樣式。
GUI（PyQt5）：
提供用戶界面，支持資料夾選擇、顏色選擇、進度條顯示和按鈕操作。
7. 修改與維護
修改主題樣式：

在 create_ppt_from_outline 函數中調整字體大小、顏色或背景樣式。
添加新功能：

可在 GUI 中添加更多選項，例如字體選擇或幻燈片過渡效果。
API 替換：

如果需要使用其他 AI 平台，可替換 get_ai_response 函數中的 API 調用邏輯。
